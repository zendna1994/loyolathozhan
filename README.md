# 🤝 Loyola Thozhan (லோயோலா தோழன்)

**Loyola Thozhan** is an AI-powered Campus Assistant designed to help 10,000+ students navigate the 24+ departments and facilities of **Loyola College, Chennai**.

![Loyola Thozhan Logo](https://zendna1994.github.io/loyolathozhan/LoyolaThozhan.jpg)

## 🚀 Vision
To provide a 100% free, lifetime-accessible digital companion that answers student queries about:
* Department Details (24 UG Departments)
* Admission Procedures
* Campus Facilities & Maps
* Exam Cell Information

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Hosted on GitHub Pages)
* **Backend:** Google Apps Script
* **Database:** Google Sheets
* **AI Brain:** Gemini 3 Flash API
